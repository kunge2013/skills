## 数据流

```
┌──────────────────┐         ┌───────────────────────────┐
│   GitHub Remote  │         │   npm Registry            │
│   (kunge2013/    │         │   (kungeskill package)    │
│    skills)       │         │                           │
└────────┬─────────┘         └─────────────┬─────────────┘
         │ git clone/pull                   │ npm install -g
         ▼                                  ▼
┌──────────────────────┐      ┌──────────────────────────┐
│  Git Cache           │      │  Bundled Plugins         │
│                      │      │                          │
│  ~/.kungeskills/     │      │  <npm-global>/           │
│  cache/marketplace/  │      │  kungeskill/plugins/     │
│                      │      │                          │
│  ├── .claude-plugin/ │      │  ├── .claude-plugin/     │
│  │   └── market-    │      │  │   └── marketplace.json│
│  │       place.json │      │  ├── openspec-workflow/   │
│  ├── plugins/        │      │  ├── openspec-trace/     │
│  │   ├── openspec-  │      │  └── publish/            │
│  │   │   workflow/  │      │                          │
│  │   ├── openspec-  │      └──────────────┬───────────┘
│  │   │   trace/     │                     │
│  │   └── publish/   │                     │
│  └── .git/           │                     │
└──────────┬───────────┘                     │
           │                                 │
           └──────────────┬──────────────────┘
                          │
                          ▼  getAllMarketplaceDirs()
              ┌───────────────────────┐
              │   Merged Skill List   │
              │                       │
              │  优先级: cache > bundled│
```

## 跨平台 Symlink 策略

```
                    createSkillSymlink(source, link)
                              │
                              ▼
                   ┌─────────────────────┐
                   │  platform === win32? │
                   └──────┬──────────────┘
                          │
              ┌───────────┴───────────┐
              ▼                       ▼
     ┌─────────────┐        ┌─────────────────┐
     │   Windows   │        │  Linux / macOS  │
     └──────┬──────┘        └────────┬────────┘
            │                        │
            ▼                        ▼
   ┌─────────────────┐      fs.symlinkSync(
   │ 同一驱动器？     │        source, link, 'dir')
   └───┬─────────┬───┘
       │ Yes     │ No
       ▼         ▼
  junction    copyDirRecursive()
  (fs.symlink   ┌────────────────┐
   'junction')  │ 递归复制文件    │
                │ 不自动同步更新  │
                │ 需 --force 刷新 │
                └────────────────┘
```

## 项目定位逻辑

```
                    findProjectSkillsDir(cwd)
                              │
                              ▼
                 ┌────────────────────────┐
                 │  从 cwd 向上遍历查找   │
                 │  第一个含 .git 的目录   │
                 └────────────┬───────────┘
                              │
              ┌───────────────┴───────────────┐
              ▼                               ▼
     ┌────────────────┐              ┌────────────────┐
     │ 找到 .git      │              │ 未找到 .git    │
     │ projectRoot =  │              │ projectRoot =  │
     │ 该目录         │              │ cwd            │
     └───────┬────────┘              └───────┬────────┘
             │                               │
             └───────────────┬───────────────┘
                             ▼
                  <projectRoot>/.claude/skills/
                  (不存在则自动创建)
```
